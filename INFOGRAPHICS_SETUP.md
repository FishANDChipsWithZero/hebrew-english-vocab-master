# הוראות הוספת אינפוגרפיות

## מה עשינו:

✅ **נוספה קומפוננטה חדשה**: `PastTenseLearn.tsx` - מציגה 3 אינפוגרפיות במצגת אינטראקטיבית

✅ **עודכנה הכותרת**: מ-"בחרי את סוג התרגול" ל-"בחרי מה ללמוד/לתרגל"

✅ **נוספו 2 כפתורים**:
- 🕐 Past Simple / Progressive - תרגול (ורוד)
- 📖 Past Simple / Progressive - לימוד (סגול)

✅ **יצרנו תיקייה**: `public/infographics/` לקבצי האינפוגרפיקות

## איך להוסיף את האינפוגרפיות שלך:

1. **שמרי את 3 האינפוגרפיות** בשמות הבאים:
   ```
   public/infographics/past_simple.jpg
   public/infographics/past_progressive.jpg
   public/infographics/past_diff.jpg
   ```

2. **פורמט מומלץ**:
   - סוג קובץ: JPG, PNG, או GIF
   - גודל: 800x600 פיקסלים או יותר
   - איכות: גבוהה (לא דחוסה מדי)

3. **אם אין לך את התמונות עדיין**:
   - המערכת תציג placeholder אוטומטי
   - תוכלי להוסיף אותן מאוחר יותר

## התכונות החדשות:

### מצב לימוד (Learning Mode):
- מצגת אינטראקטיבית עם 3 שקופיות
- ניווט קדימה/אחורה
- נקודות אינדיקטור (dots)
- מונה שקופיות
- טיפ בסוף המצגת להמשיך לתרגול

### עיצוב:
- רספונסיבי (מתאים לכל המסכים)
- אנימציות חלקות
- עיצוב זהה לשאר האפליקציה
- כפתור "חזור" לחזרה למסך בחירה

## איך זה עובד:

1. משתמש נכנס לדף הבחירה
2. רואה את הכותרת: "בחרי מה ללמוד/לתרגל"
3. בוחר "Past Simple / Progressive - לימוד" 📖
4. עובר למצב לימוד עם האינפוגרפיות
5. קורא ולומד את החומר
6. לוחץ "חזור" וממשיך ל"תרגול" 🕐
