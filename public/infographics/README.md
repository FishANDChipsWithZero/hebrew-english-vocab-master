# אינפוגרפיות Past Simple / Progressive

תיקייה זו מיועדת לאינפוגרפיות הלימודיות.

## כיצד להוסיף את האינפוגרפיות שלך:

1. שמרי את 3 האינפוגרפיות בשמות הבאים:
   - `past_simple.jpg` - אינפוגרפיקה על Past Simple
   - `past_progressive.jpg` - אינפוגרפיקה על Past Progressive  
   - `past_diff.jpg` - אינפוגרפיקה על ההבדל בין השניים

2. העתיקי את הקבצים לתיקייה זו: `public/infographics/`

3. ניתן להשתמש בפורמטים: `.jpg`, `.jpeg`, `.png`, `.gif`

## פורמט מומלץ:
- גודל: 800x600 פיקסלים לפחות
- איכות: גבוהה (לא דחוסה מדי)
- פורמט: JPG או PNG

## אם אין תמונות עדיין:
המערכת תציג placeholder עם הכיתוב "הוסף את האינפוגרפיקה כאן"
